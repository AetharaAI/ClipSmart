   ```
   "Generate complete MiniMax-M2 integration service for ClipSmart
   including: [list specific missing features from checklist]"
   ```
3. Paste M2's error logs back to me (Claude) for debugging

### If Setup Fails:
1. Check **Quick Start Guide Step 6** (Troubleshooting)
2. Verify all API keys are correct in .env
3. Check Docker logs: `docker-compose logs`
4. Ensure ports 3000, 8000, 5432, 6379 are free
5. Paste error logs to me (Claude) with environment details

### If Tests Fail:
1. Run individual test suites to isolate issue:
   ```bash
   npm test VideoPlayer.test.tsx  # Frontend
   pytest tests/test_m2_integration.py  # Backend
   ```
2. Check if issue is M2 API (test with mock data)
3. Verify FFmpeg and OpenCV installed correctly
4. Check disk space (video processing needs 5GB+ free)

### If Video Processing Fails:
1. Test FFmpeg directly: `ffmpeg -version`
2. Check Celery worker logs: `docker-compose logs clipsmart-celery`
3. Verify input video format (should be MP4, MOV, AVI, WebM)
4. Check if M2 API quota exceeded (rate limiting)
5. Try with smaller video first (<100MB, <5min)
